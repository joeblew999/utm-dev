How It Works

This app registers the tauri-basic:// URL scheme. Opening tauri-basic://hello/world in a browser will open this app and log the URL below. On desktop, the single-instance plugin focuses the existing window.

Note: The URL scheme is registered when the app is installed (via .dmg/.msi/.deb), not during cargo tauri dev.
