## utm-dev utm install

Install UTM app or download VM ISO

### Synopsis

Install the UTM application or download a VM ISO from the gallery.

Without arguments, installs the UTM application.
With a VM key, downloads the ISO for that VM.

Examples:
  # Install UTM app
  utm-dev utm install

  # Download Windows 11 ISO
  utm-dev utm install windows-11-arm

  # Force reinstall UTM
  utm-dev utm install --force

```
utm-dev utm install [vm-key] [flags]
```
