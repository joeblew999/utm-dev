## utm-dev self build

Build utm-dev binaries for all platforms

### Synopsis

Cross-compile utm-dev for all supported architectures and generate bootstrap scripts.

Output: All artifacts are placed in .dist/ directory
- Binaries: .dist/utm-dev-<platform>
- Scripts: .dist/macos-bootstrap.sh, .dist/windows-bootstrap.ps1

Flags:
  --local      Generate scripts that use local binaries (for testing)
  --obfuscate  Use garble to obfuscate code (auto-installs if needed)

This is a LOCAL build command - it does NOT create releases or push to GitHub.

```
utm-dev self build [flags]
```

  -h, --help        help for build
      --local       Generate bootstrap scripts for local testing (uses local binaries instead of GitHub releases)
      --obfuscate   Use garble to obfuscate binaries (auto-installs garble if needed)
